# 局域网团队图书管理系统

这是一个专为小团队设计的轻量级电子书管理与阅读系统。它支持上传、管理和在线阅读 EPUB、PDF 和 TXT 格式的书籍。所有的阅读进度都会自动保存在浏览器中。

由于设计目标是局域网内部署（可能无外网），因此所有前端依赖库均已包含在项目中（或需要手动下载一次放入），无需联网加载 CDN。

## 此版本亮点

1.  **极佳的 TXT 阅读体验**：后端传输纯文本，前端自动将其转换为 EPUB 格式渲染。这意味着 TXT 书籍也拥有了章节分页、字号调整、背景切换等高级功能，彻底告别“长滚动条”阅读模式。
2.  **专业的 PDF 阅读器**：集成了 PDF.js 官方完整版 Viewer，支持缩放、侧边栏目录、搜索、打印等高级功能。
3.  **零数据库配置**：使用 SQLite 单文件数据库，无需安装 MySQL 或 PostgreSQL，开箱即用。

## 功能列表

- **书籍管理**：管理员可以上传、重命名、删除书籍。
- **格式支持**：
    - **EPUB**: 原生支持。
    - **PDF**: 使用 Mozilla PDF.js 完整版查看器。
    - **TXT**: 自动在前端转换为 EPUB 风格的分页阅读。
- **阅读进度**：自动记忆上次阅读位置（基于浏览器 LocalStorage，无需账户登录）。
- **权限管理**：简单的管理员/普通用户权限分离。
- **搜索**：支持按书名快速检索。

## 部署安装

### 1. 环境准备

确保服务器或本机已安装 Python 3.8+。

### 2. 获取代码与依赖库

如果你是直接克隆本仓库，请确保 `static/js` 下的库文件完整。

(注意：由于 `pdfjs` 文件夹较大且文件众多，如果你是初次搭建，请务必检查 `static/js/pdfjs` 是否存在。如果不存在，请前往 https://github.com/mozilla/pdf.js/releases 下载 Prebuilt 包，解压并将 `web` 和 `build` 文件夹放入 `static/js/pdfjs/` 中)。

### 3. 安装 Python 依赖

```bash
pip install -r requirements.txt
```

### 4. 启动服务

```bash
python app.py
```

服务默认运行在 `0.0.0.0:5000`。
- 本机访问：`http://localhost:5000`
- 局域网访问：`http://服务器IP:5000`

## 使用说明

### 管理员登录
- 默认入口：点击右上角 "管理员登录"。
- 默认密码：**`admin123`**
- *安全提示：如果你将在生产环境使用，请务必在 `app.py` 中修改 `ADMIN_PASSWORD_HASH` 和 `app.secret_key`。*

### 上传书籍
- 登录后进入管理后台。
- 点击“上传新书”，支持批量或单文件（取决于浏览器）。
- **注意**：TXT 文件会自动处理编码，但建议保存为 UTF-8 格式以获得最佳兼容性。
